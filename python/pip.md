# pip

## 安装
下载 [get-pip.py](https://bootstrap.pypa.io/get-pip.py)

安装 pip
```
python get-pip.py
```

安装 pip3
```
python3 get-pip.py
```

### 镜像源

### ubuntu 

在 `~/.pip/pip.conf` 文件下添加

### window

在目录 `C:\Users\BinG\pip\pip.ini` 下添加

```
[global]
index-url = http://pypi.douban.com/simple

[install]
trusted-host = pypi.douban.com
```
