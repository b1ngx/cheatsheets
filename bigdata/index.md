# Bigdata


## 大数据处理流程
- 数据采集
Flume: web 日志写入 HDFS

- 数据清洗
脏数据，不符合数据规范的数据
Spark、Hive、MapReduce 或者其他一些分布式计算框架

- 数据存储
关系型数据库和非关系型数据库：RDBMS、NoSql、Hbase

- 数据可视化
通过图表的形式展现出来：饼图、柱状图、折线图、地图等
Echarts、HUE、Zeppelin
